# QuizRx Frontend

The **QuizRx Frontend** is the public-facing web application for QuizRx.  
It includes:

- Marketing / landing pages
- User onboarding and authentication UI
- Dashboard and progress view
- Quiz / question interface and follow-up chat UI

---

## Tech Stack (planned)

- Framework: Next.js / React
- Language: TypeScript or JavaScript
- Styling: Tailwind CSS (or similar)
- API: Talks to `quizrx-backend-api`

> This section will be updated to match the actual implementation.
